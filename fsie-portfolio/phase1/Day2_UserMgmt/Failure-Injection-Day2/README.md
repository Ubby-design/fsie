# Failure Injection – Day 2: Access Control

## Overview

This lab simulates real-world **Linux access control failures** and demonstrates:

- Intentional failure injection
- Observability and troubleshooting
- Root cause analysis
- Structured remediation
- Documentation with evidence (screenshots)

---

## Scenarios Covered

1. **Permission Denied – Restricted Directory**
2. **Sudo Misconfiguration – Automation Failure**
3. **Missing Group Membership – Audit Failure**

Each scenario includes:
- Failure injection steps
- Verification commands
- Observability analysis
- Root cause
- Remediation
- Screenshot evidence
